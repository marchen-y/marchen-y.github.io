身未动,心仪远--深圳

发表于 2019-08-31 | 分类于 旅行

        2019年7月31日，我终于踏上了飞往深圳的航班，虽然今天深圳的天气由于台风的影响晚点了将近4个小时，但一点也没有打消掉我对去深圳的那份激动的心情。至于为什么会这么想去深圳呢? 可能是因为那里有一群待宰的小肥羊(如上图所示，一群一直喊着只要我去深圳就要带我吃遍深圳美食的大学同学)。哈哈哈，小肥羊，你们的狼叔叔就要来了(哇哦……)

        上边的图片其实是一张从深圳飞往杭州的机票，为什么是从深圳飞往杭州而不是从杭州飞往深圳，这里有一段小插曲也算是小遗憾吧。因为是第一次坐飞机，就提前很早选择了座位希望坐到一个靠窗的位置，但因为多次一举的选择了电子登机牌，所以就不能选择纸质登记牌，自己一直有个收藏的习惯，没有拿到纸质登记牌那一刻还是有点小失落的，月有阴晴圆缺，不完美的有可能会更加印象深刻(事后知道，就算选择了电子登机牌，如果托运行李的话，也可以拿到纸质登机牌)。

        登上飞机的那一刻，心里其实还是有点觉得遗憾，可是当飞机飞到空中，飞到云海的上层，望窗外看的那一刻，我真的心里的遗憾、阴霾全部到烟消云散。看着窗外的蓝天，看着那一朵朵飘过的白云，好美，美的不知道该怎么去形容，美的不可胜收，多希望这次飞行能够一直这样飞行下去，我就看着外边的蓝天，看着外边的云朵畅游。飞往深圳的天空云朵比较少，从深圳飞往杭州的时候，又是另一番美景，晒起来……

        当时在看到上图美景的那一刻，脑海就在想会不会那片云的里边真的会有个南天门，会有瑶池美景，会有那蟠桃盛宴，好想自己也能坐在其中吃仙桃，喝仙酒。

        “我们的孤独就像天空中漂浮的城市，仿佛是一个秘密，却无从述说”。这是著名动漫《天空之城》里边的一段台词。我在想，可能当你看到这种景色的那一刻，你也会无限的幻想那边云朵的背后到底藏着的是个什么神秘的物体，才会这样的吸引人对它的向往呢。

        因为之前看到别人拍到的飞机上的美景都是云海翻腾，那我也不能落后啊，遇到了这样的美景，不晒出来不可惜了吗。


        上边这两张美景是飞机刚降落到机场滑行时外边的景色，当时杭州还在下雨，但在飞机快要降落的那一刻，雨停止了，而且天空中出现了晚霞，真的好美，可能这就是为了我举行的欢迎式吧。可是之后再看起这几张图片，就很气当时旁边坐了一个那同事，如果身边坐的是一个喜欢的女生，我就会告诉她快看外边，这是我欢迎你来杭州送你的礼物，你喜欢吗？想想都肉麻，咦……

        有美景怎能无美食，毕竟我可是在深圳的这一个月时间涨了两斤肉呢，那一群小肥羊的表现的可真的是无可挑剔。先来一张闺蜜做的小龙虾


        怎么样，不错吧，要颜值有颜值，要味道你们也不知道，我品尝了，贼好吃的那种，我这种懒得剥虾的都吃了一盘，你就可以想想味道是多么的好吃了。当然，我不能不说我这个闺蜜上的了厅堂，下的了厨房，工作的一把手，回家的贤妻良母，真的优秀啊。

        真的很羡慕深圳人的生活，那早茶吃的像是皇家菜，一个早点就是慢慢的(满满的)一早上，真的特悠闲，特舒服，特好吃，结果就是待了四周吃了四周的早茶……

        一壶茶，那么多的菜，别提多自在了，回到杭州也是念念不往。既然吃了人家的那么多的菜，也帮人家打打招牌，上边吃的都是在一家叫“fan楼”的餐厅吃的，菜很多，等的人也很多，但是好吃值得等待(说个小秘密，10.30之前去人都很少的)。

        当然在深圳也吃了很多其他美食，但是没有拍照，所以也就不好意死说出来，但是这次在深圳吃东西真的是吃过瘾了，这里要真的感谢我的那群小肥羊了(我亲爱的大学兄弟和闺蜜)，希望下次我在去的时候你们还能这样表现哦，嘻嘻嘻。

        这次深圳之旅真的应该称得上是一个非常有意义的旅行，见到我从大学毕业后已经快三年没有见过的大学兄弟和闺蜜，坐了人生中的第一次飞机和拍了那么多的美景，吃了那么多的美食也没花钱。虽然分别总是带着太多的不舍，但是分别也是下次重逢的开始，我会一直期待下次的重逢，也希望你们在下次重逢之前能够一直开心下去，还有我想你们了，我的大学同学。

        今日解签，宜下江南。
